Perpustakaan Bersama

"Perpustakaan Bersama" is a web based project developed to support digital literacy access within the campus environment. This system allows users to search, read, and borrow books online, as well as manage library data more efficiently.

Project Goals

  • Facilitate students and lecturers in accessing book collections online.
  • Improve the efficiency of managing book data, members, and borrowing transactions.
  • Encourage a culture of digital literacy within the academic community.

Main Features

  • Book search by title, author, or category.
  • User account management (admin and member).
  • Book borrowing and return history.
  • Simple and user-friendly interface design.

Tools Used

VSCode, Flutter, Dart

DZ Game

DZ Game is an interface design exploration project for a modern gaming platform that showcases a variety of popular and new game titles in one place. The design focuses on creating an immersive user experience with an energetic and futuristic visual style.

Design Concept

  • Creates a dynamic and interactive gaming hub atmosphere.
  • Highlights dark visuals with red accents to evoke a competitive feel.
  • Provides a fast navigation experience focused on game content.

Tools Used

Figma.

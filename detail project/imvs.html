IMVS – Movie App Concept

MVS is a movie application design concept that showcases various popular and upcoming movie titles in a modern and intuitive interface. The main goal of this design is to create an immersive user experience with simple yet visually engaging navigation.

Project Description

This project focuses on UI/UX exploration using Figma, applying principles of visual hierarchy, contrast colors, and the use of high-resolution images to highlight the main content — movie posters. There is no interactive prototype for this project, as it aims solely for design exploration purposes.

Tools Used

  • Figma

Mobile Application Development Class Project

Project Description

This project is an assignment from the 2023 Mobile Application Development Practicum (PAB) course. The application was built using Flutter and Dart, featuring a login page, user profile, and a UNIQLO product catalog with a simple and easy-to-use design.

Project Goals

  • The goal of this project is to understand the flow of mobile application development, starting from interface design, page-to-page navigation logic, to displaying simple dynamic data.

Tools Used

  • Flutter, Dart, Visual Studio Code
